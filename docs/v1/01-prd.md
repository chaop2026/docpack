# DocPack v1 — Product Requirements Document

## 1. 프로젝트 개요

**DocPack**은 고용량 이미지를 압축하고 PDF로 변환해주는 웹 서비스입니다.
TeacherMatch 등 서류 제출이 필요한 플랫폼과 연동을 목표로 합니다.

- **URL**: docpack.kr (예정)
- **스택**: Rails 8 + Hotwire + PostgreSQL
- **포트**: 3001 (개발), 추후 Fly.io 배포

---

## 2. 핵심 기능

### 2-1. 이미지 → PDF 변환 (핵심)
- 여러 장의 이미지를 업로드하면 페이지 순서대로 1개의 PDF 생성
- 최종 PDF 용량 **2MB 이하** 자동 압축
- 지원 입력 형식: JPEG, PNG, HEIC, WEBP
- 페이지 순서 드래그앤드롭으로 변경 가능
- A4 기준 출력 시 충분한 선명도 유지

### 2-2. 이미지 단독 압축
- 이미지 1장 또는 여러 장을 2MB 이하로 자동 압축
- 출력 형식 선택: JPEG / PNG
- 원본 대비 압축률 표시 (예: 3.2MB → 0.8MB, 75% 절감)
- ZIP으로 일괄 다운로드

### 2-3. 소셜미디어 프리셋 (v1 후반 또는 v2)
| 플랫폼 | 비율 | 해상도 |
|--------|------|--------|
| 인스타그램 정방형 | 1:1 | 1080×1080 |
| 인스타그램 세로 | 4:5 | 1080×1350 |
| 페이스북 피드 | 1.91:1 | 1200×630 |
| 페이스북 스토리 | 9:16 | 1080×1920 |

- 선택한 프리셋에 맞게 자동 크롭 + 리사이즈
- 2MB 이하 자동 압축 적용

---

## 3. 페이지 구조

```
/                   ← 메인 (업로드 + 변환)
/compress           ← 이미지 압축 전용
/pdf                ← PDF 변환 전용
/social             ← 소셜미디어 프리셋
/about              ← 서비스 소개 + TeacherMatch 연결 안내
```

---

## 4. UX 흐름

```
이미지 업로드 (드래그앤드롭 또는 파일 선택)
        ↓
변환 방식 선택 (PDF / 이미지 압축 / 소셜 프리셋)
        ↓
자동 처리 (서버사이드 libvips)
        ↓
결과 미리보기 + 용량 비교 표시
        ↓
다운로드 버튼
```

---

## 5. 수익화

- **Google AdSense** 광고 슬롯 (상단 배너, 결과 화면 하단)
- 무료 사용량 제한 없음 (v1)
- 추후 프리미엄 플랜 고려 (배치 처리, 히스토리 저장)

---

## 6. TeacherMatch 연동 계획

- TeacherMatch 서류 업로드 페이지에서 DocPack 링크 제공
- 추후 SSO 로그인 연동 (v2)
- 모바일: PWA 설치 유도 배너

---

## 7. 비기능 요구사항

- 업로드 파일 서버에 임시 저장 후 **즉시 삭제** (개인정보 보호)
- 최대 업로드: 이미지 50장, 단일 파일 20MB
- 모바일 반응형 필수
- 한국어/영어 UI (한국어 우선)
